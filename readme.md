## gameOfLife
<a href="https://github.com/DD-exe/gameOfLife/blob/master/readme_EN.md">English Version</a><br>
项目旨在实现一图形化Windows程序，用于拓展参数的康威生命游戏的可视化模拟。<br>
在康威生命游戏的故事中，往往会强调其初始参数的重要性――例如，过弱的存活条件参数会使其退化为无序扩张，过强的存活条件会让模拟一片死寂。然而，大多数可用的生命游戏模拟器不允许自由调整这些参数，限制了用户探索和理解系统行为的能力。<br>
在本项目中，我们引入了对生命游戏参数的调整功能，并让其以一可接受的代价进行。现在，模拟不再局限于经典的“最佳”参数――您可以随时修改它们，即使在模拟过程中也可以。<br>
项目同样包含其他生命游戏可视化模拟中的常见功能，例如可调地图大小、可调迭代速度。<br><br>
在项目2.0版本中，我们进一步利用高效合理的算法和代码框架，拓展康威生命游戏的新边界：<br>
我们在已有程序的基础上，考虑了易移植的跑团掷骰判定机制，创造性地发明并实现了对战版生命游戏规则。现在，多种群和演化战争的模拟变得可行了。<br>
我们基于ENet设计了项目的去中心化在线对战网络结构，和简便联机流程，并设计了联机对战的更强推演约束。显然，这是由模拟器向更具游戏性方向发展的必经之路。<br>
我们深度优化了过去的页面绘制效率瓶颈，并引入了无限网格移动、阵营颜色选择、规则介绍等用户体验导向内容。现在，您可以体验到如同专门绘图软件般的自在。<br>
我们引入了基于json序列化的数据库，让玩家便利地保存和读取存档，并开放了对其的直接修改。<br><br>
